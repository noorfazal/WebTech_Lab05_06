Renaissance of Floats

By J. Smith | Mar 16, 2026

CSS floats remain fundamental. Originally for text wrap, they became layout backbone for over a decade. The technique, though old, still offers unique layout possibilities.

The float property removes element from normal flow, positioning left/right. Content flows around it naturally, creating magazine-style layouts.

Floats work by removing element from normal flow. Content wraps around, creating magazine effect that's still useful today.

Clearfix hack using :after with clear:both contains floated children properly, solving container collapse issues.

Modern browsers fully support floats for image wrapping and multi-column text. Perfect for traditional print-like web layouts.
